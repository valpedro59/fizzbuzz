# FizzBuzz et table de multiplication

## Description. Deux classiques du développement pour maîtriser les boucles et les conditions

Durée estimée : 4-5 heures
Compétences travaillées
• Boucles for et while
• Opérateur modulo (%)
• Conditions imbriquées
• Fonctions réutilisables
Ce que le projet doit faire
• FizzBuzz : afficher les nombres de 1 à 100, mais "Fizz" pour les multiples de 3, "Buzz" pour ceux
de 5, "FizzBuzz" pour les multiples des deux
• Table de multiplication : une fonction qui affiche la table d'un nombre donné (de 1 à 10)
• Générer les tables de 1 à 9 avec une boucle
